# PSQL Database в Docker-контейнере

## Сборка и запуск контейнера

1. Перейдите в каталог `db`:  
   ```bash
   cd db
   ```

2. Выполните команду для сборки и запуска контейнера:  
   ```bash
   docker compose up -d
   ```

## Подключение к psql

1. Откройте терминал и введите команду:
   ```bash
   psql -h localhost -p 5432 -U admin -d db
   ```
   

2. Введите пароль:
   ```
   POSTGRES_PASSWORD: admin
   ```

   

## Параметры подключения к БД

Для подключения к базе данных используйте следующие параметры:
```
POSTGRES_USER: admin
POSTGRES_PASSWORD: admin
POSTGRES_DB: db
```

## Хранение данных

1. Все данные БД сохраняются в volume:  
   `postgres_data`

2. Особенности работы:
   - Данные сохраняются между перезагрузками контейнера
   - Данные сохраняются после остановки или удаления контейнера
   - Для полного удаления данных необходимо явно удалить том

> **Важно:** Для сохранения данных не удаляйте том `postgres_data`
